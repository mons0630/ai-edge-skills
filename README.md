# AI Edge Gallery Skills

Custom skills for [Google AI Edge Gallery](https://github.com/google-ai-edge/gallery).

Hosted via GitHub Pages — load any skill by pointing the Gallery app at its folder URL.

## Available Skills

| Skill | Type | Description |
|---|---|---|
| [web-fetch](./web-fetch/) | JS + API | Fetch the text content of any URL |
| [web-search](./web-search/) | JS + API | Search the internet using DuckDuckGo Instant Answer API |

## How to Load

1. Open AI Edge Gallery → Agent Skills
2. Tap the Skills chip → (+) → **Load skill from URL**
3. Enter one of the URLs below:

```
https://mons0630.github.io/ai-edge-skills/web-fetch
https://mons0630.github.io/ai-edge-skills/web-search
```

> **Note**: Make sure GitHub Pages is enabled for this repo (Settings → Pages → Deploy from main branch).
